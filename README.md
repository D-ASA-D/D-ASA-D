## Profile

I am a third-year Software Engineering student at [Ryazan State Radio Engineering University](https://rsreu.ru/en) and a Backend Java student at T-Academy.

My main direction is Java backend development. I have been working with Java for about two years, with more than 1.5 years of practical experience in Spring Boot, REST APIs, databases, security, testing and deployment-oriented backend systems.

Before focusing on Java, I worked at Research Institute “Photon”, where I developed a convolutional neural network for image processing tasks.

In my free time, I work on Braverto - a small personal project where I explore ideas related to IT products, software development and digital services, with the potential to grow into a startup in the future.
